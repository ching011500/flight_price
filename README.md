# flight_data

專題主題：基於機器學習的機票價格預測

1. 收集資料：使用爬蟲技術爬取 google flights 上的機票價格資料
** 總共收集12個城市兩個月（2025/01/20-2025/03/20）的航班資料各90筆歷史價格
** 耗時龐大故使用 github actions 自動化定時爬取

2. 資料前處理：將爬取的資料進行清洗、整理、轉換成模型可用的格式
** 資料清洗：去除缺失值、重複值
** 資料整理：將資料轉換成模型可用的格式（對資料變數進行分類）
** 資料轉換：將資料轉換成適合模型使用的格式（取對數）

3. 敘述統計：對資料進行敘述性統計分析
** 資料分布：對資料進行分布分析（Histogram）
** boxplot：對資料依照變數進行盒鬚圖分析（Boxplot）
** 資料相關性：對資料進行相關性分析（Correlation）

4. 模型建立：使用機器學習模型進行機票價格預測（分艙等）
** 模型選擇：選擇適合的機器學習模型（Random Forest、SVR、XGBoost）
** 模型訓練：將資料分為訓練集和測試集進行模型訓練（7:3）
** 模型評估：使用 MSE、R2 進行模型評估

5. 效能指標：使用模型進行預測，並計算模型的效能指標
** 效能 = (隨機購買平均價格 - 預測最低價格) / 隨機購買平均價格
** 隨機購買平均價格：模仿民眾購買隨機挑選的價格
** 預測最低價格：模型預測的最低價格 
** 效能指標：可以反映節省的價格百分比
