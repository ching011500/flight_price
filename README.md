# ✈️ Flight Data — 基於機器學習的機票價格預測

本專案以機器學習方法預測機票價格，並建立互動式網頁查詢系統。  
資料來源為 Google Flights，涵蓋 12 個城市，收集 2025/01/20 至 2025/03/20 之航班價格。

---

## 📥 1. 資料收集
- 使用爬蟲技術自動化爬取 Google Flights 上的機票價格資料
- 收集 12 個城市、兩個月期間（2025/01/20-2025/03/20），每航班約 90 筆歷史價格
- 因資料量龐大，透過 GitHub Actions 定時自動爬取資料

## 🧹 2. 資料前處理
- 資料清洗：去除缺失值與重複值
- 資料整理：將變數分類，轉換為模型可用格式
- 資料轉換：針對價格欄位取對數，以符合模型假設

## 📊 3. 敘述統計分析
- 資料分布分析：繪製直方圖（Histogram）
- 變數比較分析：繪製盒鬚圖（Boxplot）
- 相關性分析：計算與視覺化變數間的相關係數（Correlation）

## ⚙️ 4. 機器學習模型建立
- 模型選擇：Random Forest、SVR、XGBoost
- 資料切分：訓練集與測試集 7:3
- 模型訓練與評估：以 MSE、R² 作為評估指標

## 🎯 5. 效能指標計算
- 效能公式：  
  $$
  \text{效能} = \frac{\text{隨機購買平均價格} - \text{預測最低價格}}{\text{隨機購買平均價格}}
  $$
- 隨機購買平均價格：模仿一般民眾隨機購買機票價格
- 預測最低價格：使用模型預測出的最低價格
- 效能指標可反映節省金額的百分比

## 📈 6. 預測價格與區間
- 根據模型表現（MSE、R²）選擇最佳模型
- 使用 Bootstrap 方法預測機票價格區間
- 提供更穩健的預測結果，並量化不確定性

## 🌐 7. Streamlit 最終成果展示
- 互動式篩選條件：
  - 出發機場、抵達機場、艙等、出發時段、航空聯盟
  - 長程航班額外提供「停靠站數量」篩選
- 結果呈現：
  - 顯示符合條件的多種航班組合
  - 顯示每筆航班的預測價格區間（下界 ~ 上界）

## 🚀 技術摘要
- Python：爬蟲、資料前處理、機器學習建模
- GitHub Actions：自動化資料收集
- Streamlit：網頁前端互動式系統
- 機器學習模型：Random Forest、SVR、XGBoost
- 資料處理與視覺化：Pandas、Matplotlib、Seaborn
